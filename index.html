Controle Financeiro com Gráfico

Este projeto é uma aplicação de controle financeiro pessoal desenvolvida em Windows Forms (WinForms) com as seguintes funcionalidades principais:

✅ Botões funcionais: Permitem adicionar, remover, listar e atualizar os dados no sistema de forma dinâmica e intuitiva.

✅ Exibição dos dados em um DataGridView: Todas as transações (receitas e despesas) são listadas em uma tabela, com informações como descrição, valor, tipo e data.

✅ Remoção de itens: O usuário pode remover facilmente a linha selecionada no DataGridView, atualizando automaticamente os dados e o gráfico.

✅ Gráfico de barras interativo: Os valores são separados por tipo e exibidos visualmente.
